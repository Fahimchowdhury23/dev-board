Question-1: What are the different ways to select an element in the DOM?

Well, These are the five ways I've learned so far.
1. getElementById()
2. getElementsByClassName()
3. getElementsByTagName()
4. querySelector()
5. querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent?

1. innerHTML: It gets or sets the HTML content inside an element, including any HTML tags.
2. innerText: It gets or sets the visible text inside an element, excluding any HTML tags.
3. textContent: Similar to innerText, but it gets or sets all text inside an element, including hidden text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a single event listener is attached to a parent element, which listens for events on its child elements. It represent their child elements, like the meaning of the delegation itself.

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept where an event that is triggered on a child element bubbles up to its parent elements, all the way to the root of the document. There are 3 phases of it :
1. Capturing Phase,
2. Target Phase,
3. Bubble Phase.

Question-5: How do you create, add, and remove elements using JavaScript?

1. Create: document.createElement(tagName). [The tag name should be in a "string"]
2. Add: parentElement.appendChild(childElement).
3. Remove: childElement.remove() or parentElement.removeChild(childElement).
